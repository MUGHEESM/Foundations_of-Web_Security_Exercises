# Acceptable Use Policy (Personal Lab)

## Hard Boundaries
- Attack **only** services you run in this repository.
- Target only `localhost` / `127.0.0.1`.
- Do not scan, probe, or exploit external systems.
- Keep vulnerable containers isolated to local Docker network.

## Practice Rules
- Treat all vulnerabilities as educational only.
- Keep exploit code local and private.
- Reset your environment after each week.

## Allowed Tooling
- Local proxy/interception tools
- Local fuzzing tools against this lab only
- Local SQL/browser/dev tools

## Not Allowed
- Any internet-facing attack activity
- Credential stuffing/bruteforce against non-lab systems
- Reusing lab payloads on real systems
